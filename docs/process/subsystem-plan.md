# Subsystem Plan

Planned subsystems for the agentic dev support harness. Each entry is a name and a single-sentence description.

| Subsystem | Description |
|---|---|
| Code Retrieval | AST-aware chunking and BM25/embedding hybrid retrieval for locating relevant application code without loading whole files. |
| Agent Feature Planning | Skill-driven workflow for planning new features, with optional volatility-decomposition for work that crosses architectural boundaries. |
| Agent Refactor Planning | Skill-driven workflow for scoping and safely executing targeted refactorings, including blast radius analysis and test coverage checks. |
